# grid-resource-wars
A multi-agent reinforcement learning strategy environment for Kaggle Game Arena.

## Starter Agent
A simple baseline agent is provided in starter_agent.py which moves toward the nearest visible resource.

## Environment
environment.py contains the grid world simulation used for evaluation.

## Quick Test

You can simulate a simple match locally.

Create a file named test_run.py and run:

python test_run.py

## Example Output

Running `python test_run.py` produces output like:

Final Scores:
Agent 0 Score: 3
Agent 1 Score: 2
Agent 2 Score: 1
Agent 3 Score: 4
